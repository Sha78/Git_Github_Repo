Git and Github

  1. Create folder(directory) on the computer
  2. Go into folder you just created and create a file(index.html)
  3. Go outside of folder and left-click then open folder with text editor
  4. Open Text Editor and make changes to the index.html
  5. Go to location of Folder you just created > left-click and open git bash
  6. Initialize the folder in git bash with git init comman
  7. git config --global user.name "Your Name"
  8. git config --global user.email "Your_Email@email.com"
  9. git add index.html to add index.html to staging area
  10. git status to see what is in the staging area
  11. To remove a file: git rm --cached index.html
  12. git commit : This will open vim editor, you need to press i in order go into insert mode. Then press Esc and type :wq and enter in order to close vim
  13. git commit -m "message" will allow you to commit a file without going to vim
  14. If you edit a file it will be unstaged and you need to add with git add .
  15. You can create a file called gitignore to create a file that does not show.
  16. Go to github and create a new repository
  17. copy the HTTPS link and go back to git bash
  18. type git remote add origin and paste the HTTPS link ie. git remote add origin https://github.com/GitUser/Git_Github_Repo.git
  19. type git push -u origin master in git bash to push the files and directories you made into Github repo
  20. create a README.md in gitbash
  21. Go into editor and edit README.md file
  22. git add .
  23. git commit -m "message"
  24. git push
