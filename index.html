Git and Github

  1. Create folder(directory) on the computer
  2. Go into folder you just created and create a file(index.html)
  3. Go outside of folder and left-click then open folder with text editor
  4. Open Text Editor and make changes to the index.html
  5. Go to location of Folder you just created > left-click and open git bash
  6. Initialize the folder in git bash with git init command
